Listes des commandes du menu Lpx ou Scénario

 

Le menu Lpx propose les commandes suivantes :

 

Compile

Compile un scénario LPX.

Compile Tout

Compilation de tous les scénarios non compilés

 

Recompile Tout

Recompilation de tous les scénarios LPX et Graphiques.

 

Execute

Exécute un scénario LPX.

 

 

 

Le menu Scénario propose les commandes suivantes :

 

Execute

Exécute un scénario graphique.

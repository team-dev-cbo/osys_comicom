Commande Coller

Menu "Edition/Coller"

 

Utilisez cette commande pour insérer une copie du contenu du Presse-papiers au point d'insertion. Cette commande n'est pas disponible si le Presse-papiers est vide.

Pour l'élément fenêtre, on colle un objet graphique.

 

Raccourcis ( sauf pour l'élément fenêtre )

Barre d'outils : 

Touches :  CTRL+V

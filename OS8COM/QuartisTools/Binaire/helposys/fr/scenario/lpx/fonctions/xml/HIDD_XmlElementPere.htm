XmlElementPere, Fonction

 

Positionne sur l’élément père de l’élément courant de la structure xml.

 

 

Syntaxe

 

XmlElementPere( p_xml : T_Xml ) RETURN Integer

 

Elément

Description

 

p_xml

identifiant structure xml

 

 

Valeur retournée

0 : OK ,

1 : erreur positionnement,

2 : erreur identifiant structure xml.

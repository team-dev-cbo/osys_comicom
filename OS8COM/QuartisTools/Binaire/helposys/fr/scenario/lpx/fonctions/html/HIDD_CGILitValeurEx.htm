CGILitValeurEx, Fonction

 

Lecture du contenu d’un elément du formulaire. Le contenu de cet élément est stocké dans une chaine de très grande taille ( 65000 caractères maximum ).

 

 

Syntaxe

 

CGILitValeurEx( p_nom : GrandeChaine , VAR p_chaine ChaineEx )

 

Elément

Description

 

p_nom

Nom de l’élément du formulaire

p_chaine

Chaine de réception du contenu

 

 

Exemple

 

WITH  Html ;

 

PROCEDURE Exemple ;

 

VAR

 l_chaine : ChaineEx ;

 

BEGIN

 

 // début de la page HTML

 HtmlPageDebut( "Exemple","",0 );

 

 // début du formulaire

 HtmlFormDebut( "SFAO") ;

 

 // définition d'une zone de Texte

 HtmlZoneTexte( "zone2" , "texte initial", 4,100, TRUE ) ;

 

 // bouton d'envoi des données

 HtmlBoutonValide( "Bouton" , "Validation" ) ;

 

 // fin du formulaire

 HtmlFormFin ;

 

 // fin de la page HTML

 HtmlPageFin;

 

 // Attente de la saisie

 AttenteSaisie ;

 

 // Lecture de la saisie

 CGILitValeurEx( "zone2", @ l_chaine );

 

 

 

END Exemple ;

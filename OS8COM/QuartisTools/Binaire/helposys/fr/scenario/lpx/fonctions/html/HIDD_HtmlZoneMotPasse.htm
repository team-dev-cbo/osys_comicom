HtmlZoneMotPasse, Fonction

 

Définit une zone dans un formulaire où les caractères saisis sont cachés et représentés par le caractère *.

 

 

Syntaxe

 

HtmlZoneMotPasse( p_nom,p_valeur  : GrandeChaine ; p_taille : Integer )

 

 

Elément

Description

 

p_nom

Nom de la zone

p_valeur  

Contenu initial de la zone.

p_taille

Taille de la zone visible à l'écran ( en nombre de caractères )

 

 

Exemple

 

WITH Html ;

 

PROCEDURE Exemple ;

 

VAR

 l_chaine : GrandeChaine ;

 

BEGIN

 

 // début de la page HTML

 HtmlPageDebut( "Exemple" ,"",0);

 

 // début du formulaire

 HtmlFormDebut( "SFAO") ;

 

 // définition d'une zone de saisie

 HtmlZoneMotPasse( "zone1" , "texte initial" , 20 ) ;

 

 // bouton d'envoi des données

 HtmlBoutonValide( "Bouton" , "Validation" ) ;

 

 // fin du formulaire

 HtmlFormFin ;

 

 // fin de la page HTML

 HtmlPageFin;

 

 // Attente de la saisie

 AttenteSaisie ;

 

 // Lecture de la saisie

 l_chaine := CGILitValeur( "zone1" );

// page de réponse

 // début de la page HTML

 HtmlPageDebut( "Reponse" ,"",0 );

 

 // réponse

 HtmlTexteASCII( "Texte saisi : " + l_chaine , ROUGE );

 

 // fin de la page HTML

 HtmlPageFin;

 

 

END Exemple ;

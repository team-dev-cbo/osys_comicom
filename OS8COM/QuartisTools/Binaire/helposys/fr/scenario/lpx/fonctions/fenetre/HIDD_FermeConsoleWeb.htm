FermeConsoleWeb, Procédure

Fermeture d'une page web via l'url ou le nom de la console passée en paramètre.

Syntaxe

FermeConsoleWeb (URL : TresGrandeChaine )

Elément

Description

 

URL

Nom de la page web à fermer  ou l'url entière.

 

 

Exemple

 

With stdlib;

 

   // Fermetuer d'une page web

 

    FermeConsoleWeb("Nom de la page Web" );

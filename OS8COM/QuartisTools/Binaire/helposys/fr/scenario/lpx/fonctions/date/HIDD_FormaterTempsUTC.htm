FormaterTempsUTC, Fonction

Conversion d’une date+heure (stockée dans un entier) en chaîne formatée pour l'UTC (0).

L' UTC (0) est un fuseau horaire, sans décalage horaire par rapport à l'UTC (Temps universel coordonné).

Syntaxe

FormaterTempsUTC (Format : GrandeChaine; Date : INTEGER) return GrandeChaine

 

Elément

Description

 

Format

Format de présentation de la date

Date

Contient le nombre de secondes écoulé depuis le 1er janvier 1970.

 

Format

Description

 

%a

Retourne le nom en abrégé du jour de la semaine

%A

Retourne le nom en clair du jour de la semaine

%b

Retourne le nom du mois en abrégé

%B

Retourne le nom du mois en clair

%d

Retourne le n° du jour (01-31)

%H

Retourne l'heure sur 24 h (00-23)

%I

Retourne l'heure sur 12 h (00-12)

%j

Retourne le n° du jour de l'année (001-366)

%m

Retourne le n° du mois (01-12)

%M

Retourne les minutes (00-59)

%S

Retourne les secondes (00-59)

%U

Retourne le n° de semaine dans l'année.( 00 – 53 )

%w

Retourne le n° du jour dans la semaine (dimanche=0)

%y

Retourne l'année sur 2 chiffres (00-99)

%Y

Retourne l'année sur 4 chiffre (ex: 1993)

 

Exemple

WITH stdlib;

Procedure Exemple;

VAR

    TempsFormate : GrandeChaine ;

Begin

   TempsFormate := FormaterTempsUTC ( "%d/%m/%y %H:%M", TempsSysteme );

End Exemple;

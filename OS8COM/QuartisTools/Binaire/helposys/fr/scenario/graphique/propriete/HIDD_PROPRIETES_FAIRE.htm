Propriétés de l'action de type Faire ... Tant Que

 

Une action Faire ... Tant Que est définie par :

 

Nom

Nom de l'action Faire ... tant que.

Description

Description de l'action Faire ... Tant Que.

Mémoire testée

Nom de la mémoire contenant la première valeur.

Opérateur

Opérateur de comparaison :

<   :  Inférieur

<= : Inférieur ou égale

>   : Supérieur

>= : Supérieur ou égale

<> : Différent

 =  : Egale  

Valeur à comparer

La valeur de comparaison peut être une chaîne de caractères, un nombre ou le nom d'une mémoire.

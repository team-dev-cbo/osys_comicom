Niveau de la trace pour les différents modules

 

Indiquer le niveau de traçabilité pour chaque module. Le niveau 1 correspond à un minimum de messages et le niveau 4 à tous les messages. Les messages d'erreur  (affichés en rouge dans la fenêtre d'affichage de la trace) sont toujours affichés quelque soit le niveau des modules.

 

Pour ajouter un nouveau module, sélectionnez une ligne et cliquer sur le bouton droit de la souris. Une fenêtre apparaît avec toute la liste des modules. Sélectionnez en un. Il est automatiquement ajouté dans la liste. Puis modifié le niveau de la trace si nécessaire.

 

 

Remarque :

Module OsysSrv : Messages propre au serveur ou à des messages générales pour tous les autres modules.

Pour tous les autres modules (Serveur Web, module Utilisateurs, module Mémoires, module Requêtes, module Fenêtres, module Scénarios, module Déclenchement, module Etats) : Messages propre au module.
